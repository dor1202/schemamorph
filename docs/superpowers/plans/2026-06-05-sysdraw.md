# SysDraw Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Frontend-only system-design canvas with a Minimalist ↔ Real Tools dual-view toggle, JSON-driven community tool catalog, `.sysdraw` file persistence, and full OSS repo guards.

**Architecture:** Vite + React 18 + TS SPA. React Flow (`@xyflow/react` v12) renders one custom node type for all archetypes; rendering branches on global `viewMode`/`nodeStyle` from a Zustand store with snapshot undo history. All tools/archetypes/protocols live in zod-validated JSON catalogs — adding a tool requires zero code. Persistence: `.sysdraw` JSON export/import + debounced localStorage autosave.

**Tech Stack:** Vite, React 18, TypeScript, `@xyflow/react`, Zustand, zod, Tailwind v4 (`@tailwindcss/vite`), simple-icons, `@dagrejs/dagre`, html-to-image, sonner (toasts), `@vercel/analytics`, Vitest + React Testing Library.

**Specs:** `docs/superpowers/specs/2026-06-05-sysdraw-design.md`, `docs/superpowers/specs/2026-06-05-sysdraw-repo-governance.md`

**RULES FOR EXECUTORS:**
1. **NEVER run `git commit` or `git push`.** At each "USER COMMIT CHECKPOINT", stop and tell the user what is ready to commit. The user commits.
2. TDD: write the failing test, see it fail, implement, see it pass.
3. Styling note: spec names shadcn/ui; this plan hand-rolls small Tailwind primitives in the shadcn aesthetic to avoid the interactive shadcn CLI during automated execution. shadcn components can be adopted later without design change.

```ts
import { z } from 'zod';

export const archetypeSchema = z.object({
  label: z.string().min(1),
  brandColor: z.string().regex(/^#[0-9a-fA-F]{6}$/),
  symbolViewBox: z.string().regex(/^\d+ \d+ \d+ \d+$/),
  symbolSvg: z.string().min(1), // inner SVG markup, stroke="currentColor"
  defaultTool: z.string().min(1),
});

export const toolSchema = z.object({
  archetype: z.string().min(1),
  label: z.string().min(1),
  iconSlug: z.string().min(1).optional(), // simple-icons slug; omit -> initials fallback
  svgPath: z.string().min(1).optional(),  // inline path for tools missing from simple-icons
  brandColor: z.string().regex(/^#[0-9a-fA-F]{6}$/),
});

export const archetypesFileSchema = z.record(z.string(), archetypeSchema);
export const toolsFileSchema = z.record(z.string(), toolSchema);
export const protocolsFileSchema = z.array(z.string().min(1));

export type Archetype = z.infer<typeof archetypeSchema>;
export type Tool = z.infer<typeof toolSchema>;
```

```ts
import * as simpleIcons from 'simple-icons';

export type ResolvedIcon = { path: string; viewBox: string };

/** simple-icons exports `siMysql`-style objects; slugs are lowercase alphanumeric. */
export function getSimpleIcon(slug: string): ResolvedIcon | null {
  const key = 'si' + slug.charAt(0).toUpperCase() + slug.slice(1);
  const icon = (simpleIcons as Record<string, unknown>)[key] as { path?: string } | undefined;
  return icon?.path ? { path: icon.path, viewBox: '0 0 24 24' } : null;
}

/** Two-letter badge for tools with no resolvable icon. */
export function toolInitials(label: string): string {
  const words = label.trim().split(/\s+/);
  return (words.length >= 2 ? words[0][0] + words[1][0] : label.slice(0, 2)).toUpperCase();
}
```

- [ ] **Step 5: Create `scripts/validate-catalog.ts`** (the community no-code guard; CI runs this)

```ts
import { readFileSync } from 'node:fs';
import { archetypesFileSchema, toolsFileSchema, protocolsFileSchema } from '../src/lib/catalog-schema';
import { getSimpleIcon } from '../src/lib/icons';

const errors: string[] = [];

function load(path: string): unknown {
  return JSON.parse(readFileSync(new URL(`../src/catalog/${path}`, import.meta.url), 'utf8'));
}

const archetypesResult = archetypesFileSchema.safeParse(load('archetypes.json'));
const toolsResult = toolsFileSchema.safeParse(load('tools.json'));
const protocolsResult = protocolsFileSchema.safeParse(load('protocols.json'));

for (const [name, result] of [
  ['archetypes.json', archetypesResult],
  ['tools.json', toolsResult],
  ['protocols.json', protocolsResult],
] as const) {
  if (!result.success) {
    for (const issue of result.error.issues) {
      errors.push(`${name} -> ${issue.path.join('.')}: ${issue.message}`);
    }
  }
}

if (archetypesResult.success && toolsResult.success) {
  const archetypes = archetypesResult.data;
  const tools = toolsResult.data;
  for (const [key, tool] of Object.entries(tools)) {
    if (!archetypes[tool.archetype]) {
      errors.push(`tools.json -> ${key}: unknown archetype "${tool.archetype}"`);
    }
    if (tool.iconSlug && !getSimpleIcon(tool.iconSlug)) {
      errors.push(`tools.json -> ${key}: iconSlug "${tool.iconSlug}" not found in simple-icons (remove it or use svgPath)`);
    }
  }
  for (const [key, a] of Object.entries(archetypes)) {
    if (!tools[a.defaultTool]) {
      errors.push(`archetypes.json -> ${key}: defaultTool "${a.defaultTool}" not in tools.json`);
    } else if (tools[a.defaultTool].archetype !== key) {
      errors.push(`archetypes.json -> ${key}: defaultTool "${a.defaultTool}" belongs to a different archetype`);
    }
  }
}

if (errors.length) {
  console.error(`Catalog validation failed:\n${errors.map((e) => `  ✗ ${e}`).join('\n')}`);
  process.exit(1);
}
console.log('Catalog valid ✓');
```

```ts
} from '@xyflow/react';
import type { SysNode, SysEdge, ViewMode, NodeStyle, LayoutDirection, SysNodeData } from '@/lib/types';
import { getTool } from '@/lib/catalog';
import { HISTORY_CAP } from '@/config';

type Snapshot = { nodes: SysNode[]; edges: SysEdge[] };

export type StoreState = {
  nodes: SysNode[];
  edges: SysEdge[];
  viewMode: ViewMode;
  nodeStyle: NodeStyle;
  layoutDirection: LayoutDirection;
  past: Snapshot[];
  future: Snapshot[];

  onNodesChange: (changes: NodeChange<SysNode>[]) => void;
  onEdgesChange: (changes: EdgeChange<SysEdge>[]) => void;
  onConnect: (connection: Connection) => void;
  setNodes: (nodes: SysNode[]) => void;
  setEdges: (edges: SysEdge[]) => void;

  addNode: (archetype: string, concreteTool: string, position: { x: number; y: number }) => void;
  updateNodeData: (id: string, data: Partial<SysNodeData>) => void;
  updateEdgeData: (id: string, data: Partial<NonNullable<SysEdge['data']>>) => void;
  deleteSelection: () => void;
  duplicateSelection: () => void;
  selectAll: () => void;
  nudgeSelection: (dx: number, dy: number) => void;
  applyPositions: (positions: Map<string, { x: number; y: number }>) => void;

  setViewMode: (m: ViewMode) => void;
  toggleViewMode: () => void;
  setNodeStyle: (s: NodeStyle) => void;
  setLayoutDirection: (d: LayoutDirection) => void;

  snapshot: () => void;
  undo: () => void;
  redo: () => void;
  setAll: (nodes: SysNode[], edges: SysEdge[], meta?: { viewMode?: ViewMode; nodeStyle?: NodeStyle }) => void;
  reset: () => void;
};

const clone = <T,>(v: T): T => structuredClone(v);

export const useStore = create<StoreState>((set, get) => ({
  nodes: [],
  edges: [],
  viewMode: 'minimalist',
  nodeStyle: 'card',
  layoutDirection: 'LR',
  past: [],
  future: [],

  snapshot: () => {
    const { nodes, edges, past } = get();
    set({ past: [...past, clone({ nodes, edges })].slice(-HISTORY_CAP), future: [] });
  },

  undo: () => {
    const { past, future, nodes, edges } = get();
    const prev = past[past.length - 1];
    if (!prev) return;
    set({
      nodes: prev.nodes, edges: prev.edges,
      past: past.slice(0, -1),
      future: [...future, clone({ nodes, edges })],
    });
  },

  redo: () => {
    const { past, future, nodes, edges } = get();
    const next = future[future.length - 1];
    if (!next) return;
    set({
      nodes: next.nodes, edges: next.edges,
      future: future.slice(0, -1),
      past: [...past, clone({ nodes, edges })],
    });
  },

  onNodesChange: (changes) => set({ nodes: applyNodeChanges(changes, get().nodes) }),
  onEdgesChange: (changes) => set({ edges: applyEdgeChanges(changes, get().edges) }),
  onConnect: (connection) => {
    get().snapshot();
    set({ edges: addEdge({ ...connection, type: 'sysEdge', data: {} }, get().edges) });
  },
  setNodes: (nodes) => set({ nodes }),
  setEdges: (edges) => set({ edges }),

  addNode: (archetype, concreteTool, position) => {
    get().snapshot();
    const tool = getTool(concreteTool);
    const node: SysNode = {
      id: crypto.randomUUID(),
      type: 'sysNode',
      position,
      data: { archetype, concreteTool, label: tool?.label ?? concreteTool },
    };
    set({ nodes: [...get().nodes, node] });
  },

  updateNodeData: (id, data) => {
    get().snapshot();
    set({
      nodes: get().nodes.map((n) => (n.id === id ? { ...n, data: { ...n.data, ...data } } : n)),
    });
  },

  updateEdgeData: (id, data) => {
    get().snapshot();
    set({
      edges: get().edges.map((e) => (e.id === id ? { ...e, data: { ...e.data, ...data } } : e)),
    });
  },

  deleteSelection: () => {
    const { nodes, edges } = get();
    const doomed = new Set(nodes.filter((n) => n.selected).map((n) => n.id));
    const doomedEdges = new Set(edges.filter((e) => e.selected).map((e) => e.id));
    if (!doomed.size && !doomedEdges.size) return;
    get().snapshot();
    set({
      nodes: nodes.filter((n) => !doomed.has(n.id)),
      edges: edges.filter((e) => !doomedEdges.has(e.id) && !doomed.has(e.source) && !doomed.has(e.target)),
    });
  },

  duplicateSelection: () => {
    const selected = get().nodes.filter((n) => n.selected);
    if (!selected.length) return;
    get().snapshot();
    const copies: SysNode[] = selected.map((n) => ({
      ...clone(n),
      id: crypto.randomUUID(),
      position: { x: n.position.x + 24, y: n.position.y + 24 },
      selected: true,
    }));
    set({
      nodes: [...get().nodes.map((n) => ({ ...n, selected: false })), ...copies],
    });
  },

  selectAll: () =>
    set({
      nodes: get().nodes.map((n) => ({ ...n, selected: true })),
      edges: get().edges.map((e) => ({ ...e, selected: true })),
    }),

  nudgeSelection: (dx, dy) =>
    set({
      nodes: get().nodes.map((n) =>
        n.selected ? { ...n, position: { x: n.position.x + dx, y: n.position.y + dy } } : n
      ),
    }),

  applyPositions: (positions) => {
    get().snapshot();
    set({
      nodes: get().nodes.map((n) => {
        const p = positions.get(n.id);
        return p ? { ...n, position: p } : n;
      }),
    });
  },

  setViewMode: (viewMode) => set({ viewMode }),
  toggleViewMode: () => set({ viewMode: get().viewMode === 'minimalist' ? 'real' : 'minimalist' }),
  setNodeStyle: (nodeStyle) => set({ nodeStyle }),
  setLayoutDirection: (layoutDirection) => set({ layoutDirection }),

  setAll: (nodes, edges, meta) =>
    set({
      nodes, edges, past: [], future: [],
      ...(meta?.viewMode ? { viewMode: meta.viewMode } : {}),
      ...(meta?.nodeStyle ? { nodeStyle: meta.nodeStyle } : {}),
    }),

  reset: () =>
    set({ nodes: [], edges: [], past: [], future: [], viewMode: 'minimalist', nodeStyle: 'card', layoutDirection: 'LR' }),
}));
```

```ts
import { z } from 'zod';
import type { SysNode, SysEdge, ViewMode, NodeStyle } from './types';

export const SYSDRAW_VERSION = '1.0.0';

const positionSchema = z.object({ x: z.number(), y: z.number() });

const nodeSchema = z.object({
  id: z.string().min(1),
  type: z.literal('sysNode').catch('sysNode'),
  position: positionSchema,
  data: z.object({
    archetype: z.string().min(1),
    concreteTool: z.string().min(1),
    label: z.string(),
    customProperties: z.record(z.string(), z.string()).optional(),
  }),
});

const edgeSchema = z.object({
  id: z.string().min(1),
  source: z.string().min(1),
  target: z.string().min(1),
  type: z.literal('sysEdge').catch('sysEdge'),
  data: z.object({ label: z.string().optional(), protocol: z.string().optional() }).optional(),
});

const metaSchema = z.object({
  title: z.string().optional(),
  lastModified: z.string().optional(),
  viewMode: z.enum(['minimalist', 'real']).optional(),
  nodeStyle: z.enum(['symbol', 'card', 'plate']).optional(),
}).catch({});

export const sysdrawFileSchema = z.object({
  version: z.string(),
  meta: metaSchema.default({}),
  nodes: z.array(nodeSchema),
  edges: z.array(edgeSchema),
});

export type SysdrawFile = z.infer<typeof sysdrawFileSchema>;

export function serializeSysdraw(state: {
  nodes: SysNode[]; edges: SysEdge[]; viewMode: ViewMode; nodeStyle: NodeStyle; title?: string;
}): string {
  // strip runtime-only fields (selected, measured, dragging) down to the schema shape
  const file: SysdrawFile = {
    version: SYSDRAW_VERSION,
    meta: {
      title: state.title ?? 'architecture',
      lastModified: new Date().toISOString(),
      viewMode: state.viewMode,
      nodeStyle: state.nodeStyle,
    },
    nodes: state.nodes.map((n) => ({ id: n.id, type: 'sysNode', position: n.position, data: n.data })),
    edges: state.edges.map((e) => ({ id: e.id, source: e.source, target: e.target, type: 'sysEdge', data: e.data ?? {} })),
  };
  return JSON.stringify(file, null, 2);
}

export type ParseResult = { ok: true; data: SysdrawFile } | { ok: false; error: string };

export function parseSysdraw(text: string): ParseResult {
  let json: unknown;
  try {
    json = JSON.parse(text);
  } catch {
    return { ok: false, error: 'File is not valid JSON.' };
  }
  const result = sysdrawFileSchema.safeParse(json);
  if (!result.success) {
    const issue = result.error.issues[0];
    return { ok: false, error: `Invalid .sysdraw file — ${issue.path.join('.') || 'root'}: ${issue.message}` };
  }
  return { ok: true, data: result.data };
}
```

```ts
import { useStore } from './store';
import { serializeSysdraw, parseSysdraw, type SysdrawFile } from '@/lib/sysdraw-file';
import { AUTOSAVE_KEY } from '@/config';

const DEBOUNCE_MS = 500;

export function loadAutosave(): SysdrawFile | null {
  const text = localStorage.getItem(AUTOSAVE_KEY);
  if (!text) return null;
  const result = parseSysdraw(text);
  if (!result.ok) {
    localStorage.removeItem(AUTOSAVE_KEY); // corrupt -> discard, start fresh
    return null;
  }
  return result.data;
}

export function clearAutosave(): void {
  localStorage.removeItem(AUTOSAVE_KEY);
}

/** Subscribe to store changes; debounced write. Returns unsubscribe. */
export function startAutosave(): () => void {
  let timer: ReturnType<typeof setTimeout> | undefined;
  const unsubscribe = useStore.subscribe((state, prev) => {
    if (
      state.nodes === prev.nodes && state.edges === prev.edges &&
      state.viewMode === prev.viewMode && state.nodeStyle === prev.nodeStyle
    ) return;
    clearTimeout(timer);
    timer = setTimeout(() => {
      const { nodes, edges, viewMode, nodeStyle } = useStore.getState();
      try {
        localStorage.setItem(AUTOSAVE_KEY, serializeSysdraw({ nodes, edges, viewMode, nodeStyle }));
      } catch {
        /* quota exceeded — autosave is best-effort */
      }
    }, DEBOUNCE_MS);
  });
  return () => {
    clearTimeout(timer);
    unsubscribe();
  };
}

/** Hydrate store from autosave at app boot. */
export function restoreAutosave(): boolean {
  const saved = loadAutosave();
  if (!saved) return false;
  useStore.getState().setAll(saved.nodes as never, saved.edges as never, {
    viewMode: saved.meta.viewMode,
    nodeStyle: saved.meta.nodeStyle,
  });
  return true;
}
```

```ts
import dagre from '@dagrejs/dagre';
import type { SysNode, SysEdge, LayoutDirection } from './types';

const FALLBACK_WIDTH = 180;
const FALLBACK_HEIGHT = 60;

/** Pure layout function — community can add other engines alongside. */
export function layoutPositions(
  nodes: SysNode[],
  edges: SysEdge[],
  direction: LayoutDirection = 'LR'
): Map<string, { x: number; y: number }> {
  const g = new dagre.graphlib.Graph();
  g.setGraph({ rankdir: direction, nodesep: 60, ranksep: 90 });
  g.setDefaultEdgeLabel(() => ({}));

  for (const node of nodes) {
    g.setNode(node.id, {
      width: node.measured?.width ?? FALLBACK_WIDTH,
      height: node.measured?.height ?? FALLBACK_HEIGHT,
    });
  }
  for (const edge of edges) g.setEdge(edge.source, edge.target);

  dagre.layout(g);

  return new Map(
    nodes.map((node) => {
      const { x, y, width, height } = g.node(node.id);
      // dagre returns centers; React Flow positions are top-left
      return [node.id, { x: x - width / 2, y: y - height / 2 }];
    })
  );
}
```

```ts
import { useCallback } from 'react';
import { toPng } from 'html-to-image';
import { getNodesBounds, getViewportForBounds, useReactFlow } from '@xyflow/react';
import { toast } from 'sonner';
import { useStore } from '@/state/store';
import { serializeSysdraw, parseSysdraw, type ParseResult } from '@/lib/sysdraw-file';
import { MAX_IMPORT_BYTES } from '@/config';

/** Pure import core — testable without DOM file plumbing. */
export function importSysdrawText(text: string): ParseResult {
  const result = parseSysdraw(text);
  if (result.ok) {
    useStore.getState().setAll(result.data.nodes as never, result.data.edges as never, {
      viewMode: result.data.meta.viewMode,
      nodeStyle: result.data.meta.nodeStyle,
    });
  }
  return result;
}

function download(blobParts: BlobPart[], type: string, filename: string) {
  const url = URL.createObjectURL(new Blob(blobParts, { type }));
  const link = document.createElement('a');
  link.href = url;
  link.download = filename;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
  URL.revokeObjectURL(url);
}

export function useFileIO() {
  const { getNodes } = useReactFlow();

  const exportFile = useCallback((title = 'architecture') => {
    const { nodes, edges, viewMode, nodeStyle } = useStore.getState();
    download([serializeSysdraw({ nodes, edges, viewMode, nodeStyle, title })], 'application/json', `${title}.sysdraw`);
    toast.success('Exported .sysdraw');
  }, []);

  const importFile = useCallback((file: File) => {
    if (file.size > MAX_IMPORT_BYTES) {
      toast.error('File too large (max 5 MB).');
      return;
    }
    const reader = new FileReader();
    reader.onload = (e) => {
      const result = importSysdrawText(String(e.target?.result ?? ''));
      if (result.ok) toast.success('Diagram loaded');
      else toast.error(result.error);
    };
    reader.readAsText(file);
  }, []);

  const exportPng = useCallback(async (title = 'architecture') => {
    const nodes = getNodes();
    if (!nodes.length) {
      toast.error('Nothing to export.');
      return;
    }
    const bounds = getNodesBounds(nodes);
    const width = Math.min(2048, bounds.width + 160);
    const height = Math.min(2048, bounds.height + 160);
    const viewport = getViewportForBounds(bounds, width, height, 0.5, 2, 0.1);
    const el = document.querySelector<HTMLElement>('.react-flow__viewport');
    if (!el) return;
    const dataUrl = await toPng(el, {
      backgroundColor: '#0f1117',
      width,
      height,
      style: {
        width: `${width}px`,
        height: `${height}px`,
        transform: `translate(${viewport.x}px, ${viewport.y}px) scale(${viewport.zoom})`,
      },
    });
    const link = document.createElement('a');
    link.href = dataUrl;
    link.download = `${title}.png`;
    link.click();
    toast.success('PNG exported');
  }, [getNodes]);

  return { exportFile, importFile, exportPng };
}
```

```tsx
export function ConfigPanel() {
  const node = useStore((s) => s.nodes.find((n) => n.selected));
  const updateNodeData = useStore((s) => s.updateNodeData);
  const [labelDraft, setLabelDraft] = useState<string | null>(null);
  const [propKey, setPropKey] = useState('');
  const [propValue, setPropValue] = useState('');

  if (!node) return null;
  const { data } = node;

  const commitLabel = () => {
    if (labelDraft !== null && labelDraft !== data.label) updateNodeData(node.id, { label: labelDraft });
    setLabelDraft(null);
  };

  const changeArchetype = (archetype: string) => {
    const defaultTool = getArchetype(archetype)?.defaultTool ?? '';
    updateNodeData(node.id, { archetype, concreteTool: defaultTool }); // spec: archetype change resets tool
  };

  const addProperty = () => {
    if (!propKey.trim()) return;
    updateNodeData(node.id, { customProperties: { ...data.customProperties, [propKey.trim()]: propValue } });
    setPropKey('');
    setPropValue('');
  };

  const removeProperty = (key: string) => {
    const next = { ...data.customProperties };
    delete next[key];
    updateNodeData(node.id, { customProperties: next });
  };

  return (
    <aside data-testid="config-panel" className="w-52 shrink-0 overflow-y-auto border-l border-[var(--border)] bg-[var(--panel)] p-3">

        aria-label="Label"
        className={inputCls}
        value={labelDraft ?? data.label}
        onChange={(e) => setLabelDraft(e.target.value)}
        onBlur={commitLabel}
        onKeyDown={(e) => e.key === 'Enter' && commitLabel()}
      />

      <label className={labelCls} htmlFor="cfg-archetype">Archetype</label>
```

```ts
import { useEffect } from 'react';
import { useStore } from '@/state/store';

const NUDGE = 8;

function isEditableTarget(target: EventTarget | null): boolean {
  if (!(target instanceof HTMLElement)) return false;
  return target.tagName === 'INPUT' || target.tagName === 'TEXTAREA' || target.tagName === 'SELECT' || target.isContentEditable;
}

export function useKeyboardShortcuts() {
  useEffect(() => {
    const onKeyDown = (e: KeyboardEvent) => {
      if (isEditableTarget(e.target)) return;
      const store = useStore.getState();
      const mod = e.metaKey || e.ctrlKey;

      if (mod && e.key.toLowerCase() === 'z') {
        e.preventDefault();
        if (e.shiftKey) store.redo();
        else store.undo();
      } else if (mod && e.key.toLowerCase() === 'd') {
        e.preventDefault();
        store.duplicateSelection();
      } else if (mod && e.key.toLowerCase() === 'a') {
        e.preventDefault();
        store.selectAll();
      } else if (e.key === 'Delete' || e.key === 'Backspace') {
        e.preventDefault();
        store.deleteSelection();
      } else if (e.key.toLowerCase() === 'm' && !mod) {
        store.toggleViewMode();
      } else if (e.key.startsWith('Arrow')) {
        const hasSelection = store.nodes.some((n) => n.selected);
        if (!hasSelection) return;
        e.preventDefault();
        const [dx, dy] = {
          ArrowLeft: [-NUDGE, 0], ArrowRight: [NUDGE, 0], ArrowUp: [0, -NUDGE], ArrowDown: [0, NUDGE],
        }[e.key as 'ArrowLeft' | 'ArrowRight' | 'ArrowUp' | 'ArrowDown'];
        store.nudgeSelection(dx, dy);
      }
    };
    window.addEventListener('keydown', onKeyDown);
    return () => window.removeEventListener('keydown', onKeyDown);
  }, []);
}
```

```markdown
# .sysdraw File Format

Format version: **1.0.0** (independent of app releases)

A `.sysdraw` file is JSON:

| Field | Type | Notes |
|---|---|---|
| `version` | string | format version, currently `1.0.0` |
| `meta.title` | string? | diagram name, used for filenames |
| `meta.lastModified` | string? | ISO-8601 |
| `meta.viewMode` | `minimalist` \| `real`? | restored on load |
| `meta.nodeStyle` | `symbol` \| `card` \| `plate`? | restored on load |
| `nodes[]` | `{ id, type: 'sysNode', position: {x, y}, data }` | `data: { archetype, concreteTool, label, customProperties? }` |
| `edges[]` | `{ id, source, target, type: 'sysEdge', data? }` | `data: { label?, protocol? }` |

Compatibility rules:
- Unknown extra fields are stripped on import (forward compatible).
- Unknown `archetype`/`concreteTool` values load with a fallback badge — files
  survive catalog changes.
- Any breaking change bumps the major version and gets a migration note here.
```
